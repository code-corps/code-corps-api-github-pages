    create_or_update_issue(CodeCorps.GithubRepo.t, map) :: linking_result

    Finds or creates a GithubIssue using the data in a GitHub Issue payload.

    The process is as follows:

    • Search for the issue in our database with the payload data.

      • If we return a single GithubIssue, then the GithubIssue should be updated.
      • If there are no matching GithubIssue records, then a GithubIssue should be created.
