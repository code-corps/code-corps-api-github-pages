    code_corps v0.0.1 CodeCorps.Analytics.Segment

    Provides analytics tracking for Segment.com with an interface for making identify and track calls via the analytics-elixir package.

    You can read more about identify and track in Segment’s documentation.

    By default, in dev and test envrionments, this module will use CodeCorps.Analytics.InMemoryAPI which does not make a request to Segment’s REST API.

    In prod and staging environments, the module will use CodeCorps.Analytics.SegmentAPI which will make requests to Segment’s REST API.

    In your config/prod.exs you might set this like so:

    config :code_corps, :analytics, CodeCorps.Analytics.SegmentAPI
