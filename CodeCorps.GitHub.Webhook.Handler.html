    handle_supported(String.t(), String.t(), map()) :: {:ok, CodeCorps.GithubEvent.t()}

    Handles a fully supported GitHub event based on its type and action.

    The handling process consistes of 3 steps

    • create event record marked as “unprocessed”
    • mark event record as processing and handle it
    • mark event record as processed or errored depending on handling outcome

    handle_unsupported(String.t(), String.t(), map()) :: {:ok, CodeCorps.GithubEvent.t()}

    Handles an unsupported supported GitHub event.

    “unsupported” means that, while we generally support this event type, we do not yet support this specific event action.

    The process consistes of simply storing the event and marking it as “unsupported”.
