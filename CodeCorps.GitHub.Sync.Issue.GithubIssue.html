    create_or_update_issue(map(), CodeCorps.GithubRepo.t(), CodeCorps.GithubPullRequest.t() | nil) :: result()

    Creates or updates a CodeCorps.GithubIssue from a github issue API payload.

    The created record is associated to the provided CodeCorps.GithubRepo and, optionaly, to a provided CodeCorps.GithubPullRequest.

    The created record is also associated with a matched CodeCorps.GithubUser, which is created if necessary.
