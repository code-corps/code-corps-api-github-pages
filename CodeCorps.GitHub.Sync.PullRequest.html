    create_or_update_pull_request(map(), CodeCorps.GithubRepo.t()) :: linking_result()

    Finds or creates a GithubPullRequest using the data in a GitHub PullRequest payload.

    The process is as follows:

    • Search for the pull request in our database with the payload data.

      • If we return a single GithubPullRequest, then the GithubPullRequest should be updated.
      • If there are no matching GithubPullRequest records, then a GithubPullRequestshould be created.
